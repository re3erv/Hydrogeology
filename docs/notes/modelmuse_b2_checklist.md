# ModelMuse checklist for B2 (heterogeneous K)

Цель: проверить, что в MF6-модели задано неоднородное поле гидропроводности `K`.

## 1. Grid / DIS

- [ ] 1 слой, 10 строк, 10 столбцов.
- [ ] Размеры ячеек 100 x 100.
- [ ] `Top = 10`, `Bottom = 0`.

## 2. IC

- [ ] Начальный напор одинаковый (например, 5).

## 3. NPF (гидропроводность)

- [ ] Модель confined (`icelltype = 0`).
- [ ] Базовое поле `K = 10`.
- [ ] В центре задана низкопроницаемая линза:
  - [ ] строки 3..6,
  - [ ] столбцы 3..6,
  - [ ] `K = 1`.

## 4. CHD

- [ ] Левая граница: `Head = 10`.
- [ ] Правая граница: `Head = 0`.
- [ ] Верх/низ: no-flow.

## 5. Solver / Output

- [ ] Модель стартует и завершается без ошибок.
- [ ] Сохраняются head и budget файлы (`.hds`, `.cbc`).
- [ ] В `run_summary.txt` есть процент невязки баланса.

## 6. Визуальная проверка

- [ ] Есть `heads_map.png`.
- [ ] Есть `k_field_map.png`.
- [ ] На карте напоров заметно локальное искривление изолиний над/вокруг low-K зоны.
