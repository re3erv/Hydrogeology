# ModelMuse checklist for B3 (recharge `RCHA`)

Цель: проверить, что в MF6-модели подключено равномерное площадное питание (`RCHA`) и оно влияет на поле напоров.

## 1. Grid / DIS

- [ ] 1 слой, 10 строк, 10 столбцов.
- [ ] Размеры ячеек 100 x 100.
- [ ] `Top = 10`, `Bottom = 0`.

## 2. IC

- [ ] Начальный напор одинаковый (например, 5).

## 3. NPF

- [ ] Модель confined (`icelltype = 0`).
- [ ] Однородная гидропроводность `K = 10`.

## 4. CHD

- [ ] Левая граница: `Head = 10`.
- [ ] Правая граница: `Head = 0`.
- [ ] Верх/низ: no-flow.

## 5. RCHA

- [ ] Подключен пакет `RCHA`.
- [ ] Введён равномерный recharge по всей площади (по умолчанию `1e-4`).
- [ ] Recharge имеет положительное значение (питание, а не отбор).

## 6. Solver / Output

- [ ] Модель стартует и завершается без ошибок.
- [ ] Сохраняются head и budget файлы (`.hds`, `.cbc`).
- [ ] В `run_summary.txt` есть процент невязки баланса.

## 7. Визуальная проверка

- [ ] Есть `heads_map.png`.
- [ ] Есть `recharge_map.png`.
- [ ] В поле напоров есть внутренний подъём относительно ожидаемого линейного профиля без recharge.
