# ModelMuse checklist — этап B6 (многослойная модель и межслойные перетоки)

Цель: убедиться, что в 3-слойной MF6-модели корректно настроены вертикальные связи между слоями и что откачка в среднем слое формирует межслойный переток.

## 1) Открытие проекта

Откройте name-файл:

- `models/mf6/06_multilayer_leakage/mfsim.nam`

Проверьте, что проект загружается как MODFLOW 6.

## 2) Проверка геометрии DIS

- `nlay = 3`, `nrow = 10`, `ncol = 10`;
- `top = 30`;
- `botm = 20, 10, 0`;
- размеры ячеек: `delr = delc = 100`.

## 3) Проверка свойств NPF

- `icelltype = 0` (confined);
- горизонтальная проводимость по слоям: `k = [10, 2, 10]`;
- вертикальная проводимость по слоям: `k33 = [1, 0.05, 1]`.

Ожидание: слой 2 менее проницаем, поэтому вертикальная связь между верхним и нижним водоносными горизонтами ослаблена.

## 4) Проверка граничных условий

- `CHD` слева и справа заданы для **всех трёх слоёв**:
  - левый столбец: `head = 10`;
  - правый столбец: `head = 0`.
- `RCHA = 1e-4` (площадное питание сверху);
- `WEL` в ячейке `(layer=1, row=5, col=5)` с расходом `Q = -700`.

## 5) Проверка результатов

В папке B6 должны быть:

- `heads_layers_map.png`
- `vertical_leakage_map.png`
- `run_summary.txt`

Проверьте, что в `run_summary.txt` есть:

- напоры в центральной ячейке по слоям (`L1, L2, L3`);
- `Net vertical flow L1->L2`;
- `Net vertical flow L2->L3`;
- `Budget percent discrepancy`.

## 6) Интерпретация (sanity check)

- Откачка в среднем слое должна понижать напор сильнее всего в слое 2.
- Перетоки между слоями должны быть ненулевыми (величина/знак зависят от сформированного градиента).
- Невязка водного баланса должна быть близка к нулю.
